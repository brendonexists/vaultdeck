Quickstart

CLI Reference

Commands you will actually use every day.

Status + Regeneration

vaultdeck status vaultdeck regen eval "$(vaultdeck apply --regen)"

Use this flow when secrets changed and you need fresh shell exports.

Web UI Control

vaultdeck start vaultdeck ui-status vaultdeck stop

Starts and stops the local UI process.

Global Service Environment

vaultdeck global enable vaultdeck global status vaultdeck global disable

Controls the user-level environment.d integration for long-running services.

Maintenance

vaultdeck doctor vaultdeck check-update vaultdeck update

Health checks and updates for local installs.
